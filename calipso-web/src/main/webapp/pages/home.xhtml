Application Jakarta EE JSF + PrimeFaces
Bootstrap pour le layout.

Fonctionnalités métiers
  • Afficher la liste des comptes
  • Ajouter un nouveau compte
  • Afficher les détails d'un compte
  • Modifier les données d'un compte
  • Supprimer un compte
  • Exporter un rapport de compte (BIRT)
  • Afficher la liste des clients
  • Ajouter un nouveau client
  • Afficher les détails d'un client
  • Modifier les données d'un client
  • Supprimer un client
  • Export la liste des clients au format Excel (POI)

  • Mise en oeuvre de l'architecture SOA

  • Mise en oeuvre : (EJB 3)
  • Mise en place Web service (REST)
  • Mise en place Web service (SOAP)

  • Mise en place des regles de navigation
